## Overview

Implemented key improvements to Devkind's blog platform focusing on content presentation and author experience. Worked with Next.js frontend and Strapi CMS to add new components and fix existing issues.

## Key Features

- **Dynamic FAQs Component**: Added to blogs through Strapi's backend, editable by content authors
- **Fixed Markdown Table Rendering**: Resolved formatting issues in blog posts
- **Improved Responsiveness**: Optimized blog page layout for all screen sizes
- **Collapsible Table of Contents**: Automatic generation from blog headings with responsive behavior
- **Ongoing Work**: Services landing page development (not yet shipped)

## Technologies Used

- **Frontend**: Next.js, Chakra UI
- **Backend**: Strapi CMS
- **Styling**: Responsive CSS techniques

## Challenges and Learnings

- Integrating dynamic components with Strapi's content structure
- Debugging markdown rendering pipeline for tables
- Ensuring consistent behavior across desktop/mobile views
- Implementing TOC functionality without external libraries

## Outcome

- More flexible content authoring with dynamic FAQs
- Better readability of complex content through table fixes
- Improved mobile experience for blog readers
- Smoother navigation with collapsible TOC
