## Overview

Led the frontend overhaul of GradAccelerate's application portal, implementing a modern user flow and backend integrations to streamline candidate processing.

## Key Features

- **Three-Step Application Wizard**: Guided form process with progress tracking
- **Zod Validation**: Type-safe form validation with real-time error feedback
- **Freshworks CRM Integration**: Automated applicant data synchronization
- **n8n Workflow Automation**: Candidate parsing and selection pipeline
- **Mobile-First Responsive Design**: Optimized UI across all device sizes

## Technologies Used

- **Frontend**: Next.js, Tailwind CSS
- **Form Handling**: Zod validation
- **CRM**: Freshworks API
- **Automation**: n8n workflow engine

## Challenges and Learnings

1. **Complex Form Validation**
   - Implemented Zod schemas for multi-step form validation
   - Maintained state persistence between wizard steps

2. **CRM Integration**
   - Mapped form data to Freshworks CRM fields
   - Handled API rate limiting and error cases

3. **Workflow Automation**
   - Designed n8n workflows for candidate processing
   - Integrated resume parsing logic

4. **Responsive Challenges**
   - Created adaptive layouts for complex form elements
   - Optimized touch targets for mobile users

## Outcome

- Streamlined application process with clear progress tracking
- Mobile-friendly experience for on-the-go applicants
- Automated candidate evaluation pipeline
- Robust form validation reducing submission errors
